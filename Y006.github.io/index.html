博客介绍

这里是邱金羽的技术博客，这里分享我在学习中的总结与思考；分享一些有趣的电子制作；你在这里还可以找到一些有价值的电子文档资源。

博客结构

博客将被分为四大板块：

  • \(Electrical Engineering\) 电子专业课程：

    这个板块目前正在作为我个人学习电子专业基础知识的笔记本，但是未来我有意向将这个电子笔记本进行归纳和总结，梳理基础知识之间的脉络，打通知识之间的壁垒，建立起一套自己的电子专业课程，并且我希望能以易于理解的方式向目前没有学习过但有意向学习电子专业知识的同学提供尽可能多的帮助。我希望在我还是个小白的时候开始总结归纳，以避免学习过后遭受“知识的诅咒”，而且我希望能建立一个完整的思考体系，让不同的专业课程之间不再具有壁垒，而是成为一个连贯的整体。

  • 电子设计：

    我觉得自己进行一些电子设计是很酷的事情，即便只是点亮一个LED灯也是值得高兴很久的事。在这个板块下我希望尽可能详细地记录我进行电子设计时学习的知识和一些技术细节，来给你——另一个也对电子设计充满兴趣的人，一点点帮助。

  • 资料分享：

    在学习电子专业基础知识期间我收集了很多电子课程的课本，这些课本有的很容易获得，有些却不容易得到，我愿意提供给和我一样热爱电子的同学一点帮助，帮助他们免去查找资料的烦恼，而是能投入更多经历在知识的学习中。这个板块中，你可以加入我们，如果你有很棒的学习资料，学习经验，我非常期待着你的分享与加入。

  • 其他：

    总有些笔记是难以归类的，很多笔记只是我当作备忘录存在博客中，但是若其中一些文字可以帮助到你，对于我也是一件乐事。
